# basictexts

basictexts is an open-source, mobile-first web app and PWA for searching Alcoholics Anonymous literature by keyword or phrase. The project is being built as a free, searchable concordance that links users to official or freely available sources.

## What this project aims to do
- Search across multiple literature sources in one experience
- Show contextual results with source labels and citations
- Support shareable search and passage links
- Work as an installable progressive web app with offline-friendly behavior

## Recommended stack
- SvelteKit for the frontend
- TypeScript throughout
- Cloudflare Pages for hosting
- Cloudflare D1 and Workers for search and data access
- PWA tooling for installability and offline support
